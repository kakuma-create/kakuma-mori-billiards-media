# Kakuma Mori Billiards Media Page

Static portfolio page for overseas billiards media introductions.

Open `index.html` in a browser to view it.

## Photo assets

- Current gallery images are loaded from `assets/photo/`.
- Put additional portfolio photos into `assets/photo/` and add them to the
  photo grid in `index.html` if needed.
- Update `mailto:your-email@example.com` with the preferred public contact address.

The generated placeholder image is still in `assets/hero-billiards-media.png`,
but the page now uses Billiards Days credited photos.
